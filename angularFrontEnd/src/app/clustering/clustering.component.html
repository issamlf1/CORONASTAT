Clusters of countries

In this page,we attempt to group different countries using K-means clustering based on Average age, Number of tests performed for covid-19, Confirmed Cases, Death,Recovered cases from Corona virus.

To have a better visualisation of the clusters, we used PCA algorithm to decrease the number of dimensions.
